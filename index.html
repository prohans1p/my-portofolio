Website Management Asset

    Deskripsi

    Proyek Manajement Asset adalah Aplikasi berbasis web yang dirancang untuk membantu perusahaan dalam mencatat, memantau, dan mengelola asset secara efesien. fitur utamanya meliputi input data aset, sistem peminjaman barang dan pengembalian barang, status asset(tersedia/dipinjam/rusak/service), serta laporan data aset. Aplikasi ini dikembangkan menggunakan Laravel, MySQL, dan Bootstrap untuk tampilan antarmuka yang responsif dan mudah digunakan

Website ESDM

    Deskripsi

    Website ini dibuat untuk membantu pegawai dalam mengisi form internal seperti laporan kerja, absensi, dan pengajuan cuti secara digital. Semua data disimpan di database dan bisa langsung diakses oleh admin HR. Dengan sistem ini, proses administrasi jadi lebih efisien, minim kesalahan, dan menghemat waktu. Tampilan websitenya dibuat simpel tapi tetap profesional, biar nyaman digunakan setiap hari oleh karyawan di berbagai divisi.

    Github: link Web Surat

Website Peminjaman barang

    Deskripsi

    Website ini dirancang untuk mempermudah proses peminjaman dan pengembalian barang di lingkungan sekolah. Siswa maupun guru bisa mengajukan peminjaman lewat form online tanpa perlu datang langsung ke bagian administrasi. Data peminjaman tersimpan otomatis di sistem, termasuk informasi barang, tanggal peminjaman, dan status pengembalian. Admin juga bisa menambah, mengedit, dan memantau seluruh aktivitas lewat dashboard yang responsif dan mudah dipahami. Tujuannya biar proses peminjaman jadi lebih cepat, rapi, dan transparan.
